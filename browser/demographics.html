Demography of the Wikimedia development community

This study is based in the information obtained from the Git, Bugzilla and Mailing lists from Wikimedia. The charts below show information about the number of contributors (X axis) every four months (Y axis).

MLS AGING

Mailing list aging show the number of contributors sending messages each four months that are still active

MLS BIRTH

Mailing list birth shows the number of newcomers sending messages each four months

ITS AGING

ITS aging shows the number of contributors (changers) that are still active using the date of the first change performed each four months.

ITS BIRTH

ITS birth shows the number of contributors (changers) joining Bugzilla each four months.

SCM AGING

SCM aging shows the code developers (authors) that are still active using the date of the first commit each four months.

SCM BIRTH

SCM birth shows the number of the code developers (authors) joining each four months
